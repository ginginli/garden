# Garden Horizons Plant Images

This document lists all plant image sources from the official Wiki.

## Image Requirements
- **Format**: WebP or PNG (WebP preferred for smaller file size)
- **Size**: Recommended 128x128px or 256x256px
- **Type**: Both seed images and produce images

## How to Download Images

### Manual Method:
1. Visit each Wiki page listed above
2. Right-click on the seed/produce image
3. Save as WebP format with the filename specified above
4. Place in the appropriate folder

These can be copied to the new location.
